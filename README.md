# Cloud Resume

My cloud-hosted resume, based on [The Cloud Resume Challenge](https://cloudresumechallenge.dev/).

## Components

- My resume, as a static website
- Hosting on Azure Storage
- HTTPS and caching on Azure Frontdoor
- Visitor counter API on Azure Functions
- Counter values stored on Azure Cosmos DB


For more, check out my blog post: [Taking on the Azure Cloud Resume Challenge](https://www.bcbrookman.com/posts/taking-on-the-azure-cloud-resume-challenge)
